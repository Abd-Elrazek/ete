NCBI taxonomPhyloXML
************************************

The ncbi_taxonomy module provide utilities to annotated trees using NCBI Taxonomy database. 
It is fully integrated with the PhyloTree object through the tree.annotate_tree() method, but other functions are available in the ncbiquery module.  

Additionally, this module provides the ncbiquery standalone tool, which allows to translate taxids, species names, retrieve taxa information or annotate tree from the command line.
